## 🚀 5分钟快速上手

### 第1分钟：登录系统

```
1. 打开浏览器：http://localhost:3000
2. 输入用户名和密码
3. 点击"登录"
```

### 第2分钟：上传历史方案

```
1. 点击左侧"文档管理"
2. 点击"上传文档"
3. 拖拽Word/PDF文件
4. 填写文档类型和标签
5. 点击"开始上传"（等待5-10秒）
```

### 第3-4分钟：生成方案

```
1. 点击左侧"方案管理" → "创建方案"
2. 填写客户名称：XX银行
3. 填写项目名称：智能营销平台
4. 填写需求描述（至少200字，包含背景、需求、目标）
5. 点击"开始生成"（等待2-5分钟）
```

### 第5分钟：导出方案

```
1. AI生成完成后自动跳转到方案详情页
2. 快速浏览各章节内容
3. 点击"导出" → "导出为Word"
4. 下载Word文件
```

**恭喜！您已完成第一个方案生成！** 🎉

## 🔍 故障快速诊断

### 问题1：方案生成失败

**症状**：点击"开始生成"后报错

**诊断步骤**：
```
1. 检查网络连接：ping服务器
2. 检查需求描述：至少200字？
3. 等待5分钟后重试
4. 联系管理员检查AI API
```

### 问题2：上传文档失败

**症状**：文档上传卡在99%或报错

**诊断步骤**：
```
1. 检查文件大小：是否超过50MB？
2. 检查文件格式：是否为.docx/.pdf/.xlsx？
3. 检查文件是否损坏：能否在本地打开？
4. 更换文件重试
```

### 问题3：导出失败

**症状**：点击导出没反应或报错

**诊断步骤**：
```
1. 检查方案状态：是否为"已完成"？
2. 检查浏览器：是否阻止了下载？
3. 清除浏览器缓存后重试
4. 尝试导出其他格式
```

### 问题4：登录失败

**症状**：提示"用户名或密码错误"

**诊断步骤**：
```
1. 检查用户名拼写（区分大小写）
2. 检查密码（注意大小写、空格）
3. 尝试"忘记密码"功能
4. 联系管理员检查账号状态
```

### 问题5：方案生成质量差

**症状**：生成的内容不够专业或与需求不符

**改进方法**：
```
1. 上传更多高质量历史方案（至少10份）
2. 完善知识库（添加产品知识、技术方案）
3. 优化需求描述（至少300字，包含详细背景）
4. 人工完善AI生成的初稿
```

   - 补充细节和数据

   - 批量上传文档

   - 记住常用快捷键
   - 提升操作速度

4. **定期整理**
   - 归档已完成方案
