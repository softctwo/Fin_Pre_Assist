## 1. 系统概述

### 1.1 系统简介

金融售前方案辅助编写系统是一个基于AI的智能售前方案生成平台，帮助售前工程师快速生成高质量的售前技术方案、商务方案和投标文档。

**核心价值**：
- ⏱️ **提升效率**：方案生成时间从2-3天缩短到30分钟
- 🎯 **提高质量**：基于历史优秀方案和知识库生成专业内容
- 📚 **知识沉淀**：历史方案自动向量化，形成企业知识资产
- 🤖 **AI增强**：智谱GLM-4大模型提供智能写作支持

### 1.2 核心功能

| 功能模块 | 功能说明 | 适用场景 |
|---------|---------|---------|
| **文档管理** | 上传历史方案、产品手册，自动向量化 | 建立知识库 |
| **AI方案生成** | 基于需求描述自动生成完整方案 | 快速响应客户 |
| **方案编辑** | 富文本编辑器，支持图表插入 | 完善方案细节 |
| **知识库** | 管理产品知识、技术方案、最佳实践 | 增强AI生成效果 |
| **模板系统** | 创建可复用模板，支持变量替换 | 标准化输出 |
| **文档导出** | 导出Word/Excel/PDF格式 | 提交客户 |

### 1.3 使用场景

**场景一：快速响应客户RFP**
```
客户发来投标邀请 → 录入需求到系统 → AI生成初稿 
→ 人工审核完善 → 导出Word提交
时间：30分钟（传统方式需2-3天）
```

**场景二：售前知识管理**
```
收集历史优秀方案 → 批量上传到系统 → 自动向量化 
→ 形成企业知识库 → 后续方案自动引用
```

**场景三：标准化方案输出**
```
创建行业方案模板 → 每次接到项目 → 使用模板生成 
→ 填入客户信息 → 快速输出标准化方案
```

### 1.4 浏览器要求

**推荐浏览器**：
- ✅ Chrome 90+ （推荐）
- ✅ Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14+

**分辨率要求**：1366×768 或更高

## 2. 系统登录与注册

### 2.1 账号注册

**操作步骤**：

1. **访问注册页面**
   - 打开浏览器，访问 `http://localhost:3000`（本地部署）
   - 或访问公司内部地址（由管理员提供）

2. **填写注册信息**
   ```
   用户名：建议使用工号或姓名拼音（如：zhangsan）
   邮箱：工作邮箱，用于接收通知和找回密码
   密码：至少8位，包含大小写字母、数字
   确认密码：再次输入密码
   ```

3. **提交注册**
   - 点击"注册"按钮
   - 系统提示"注册成功，请登录"

4. **邮箱验证**（可选，取决于系统配置）
   - 查收验证邮件
   - 点击邮件中的激活链接
   - 账号激活成功

**注意事项**：
- 用户名一旦注册不可修改
- 密码需定期更换（建议3个月）
- 忘记密码可通过邮箱找回

### 2.2 用户登录

**操作步骤**：

1. **输入登录凭证**
   - 用户名：注册时填写的用户名
   - 密码：注册密码

2. **记住登录状态**（可选）
   - 勾选"记住我"，7天内免登录
   - 公共电脑请勿勾选

3. **点击登录**
   - 登录成功后跳转到首页仪表板

**常见问题**：

**Q: 提示"用户名或密码错误"？**
```
解决方法：
1. 检查用户名拼写（区分大小写）
2. 检查密码（注意大小写、空格）
3. 尝试"忘记密码"功能重置
```

**Q: 账号被锁定？**
```
原因：连续输错密码5次，系统自动锁定30分钟
解决方法：
1. 等待30分钟后重试
2. 或联系管理员手动解锁
```

**Q: 忘记密码怎么办？**
```
操作步骤：
1. 点击登录页"忘记密码"链接
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的重置链接
5. 设置新密码
```

## 3. 文档管理

### 3.1 上传文档

**功能说明**：上传历史售前方案、产品手册、竞品分析等文档，系统自动解析内容并向量化，加入知识库供AI生成时参考。

**支持格式**：
- 📄 Word文档：`.docx`（推荐）
- 📕 PDF文档：`.pdf`
- 📊 Excel表格：`.xlsx`（用于报价单参考）

**操作步骤**：

**步骤1：进入文档管理**
- 点击左侧菜单"文档管理"

**步骤2：上传文件**
- 点击右上角"上传文档"按钮
- 方式一：点击上传区域选择文件
- 方式二：直接拖拽文件到上传区域

**步骤3：填写文档信息**
```
文档名称：自动识别文件名，可手动修改
文档类型：选择类型（必填）
  - 方案文档：历史售前方案、技术方案
  - 产品手册：产品介绍、功能说明
  - 竞品分析：竞品对比、市场分析
  - 其他：培训资料、白皮书等
标签：多个标签用逗号分隔（选填）
  示例：银行,营销系统,AI,大数据
备注：文档说明（选填）
```

**步骤4：开始上传**
- 点击"开始上传"按钮
- 显示上传进度条
- 系统自动处理：
  1. 上传文件到服务器
  2. 解析文档内容
  3. 文本向量化
  4. 存储到ChromaDB

**步骤5：完成**
- 提示"上传成功"
- 文档出现在文档列表中
- 处理时间：通常5-10秒

**注意事项**：
- ⚠️ 单个文件不超过50MB
- ⚠️ 文档内容需为中文或英文
- ⚠️ 避免上传扫描版PDF（识别率低）
- ⚠️ 建议上传高质量的优秀方案

**最佳实践**：
```
初次使用建议：
1. 上传10-20份历史优秀方案
2. 上传公司所有产品手册
3. 上传行业解决方案白皮书
4. 合理使用标签，便于后续检索
```

### 3.2 查看文档列表

**功能说明**：浏览所有已上传的文档，支持搜索、筛选和排序。

**操作说明**：

**搜索文档**：
- 在搜索框输入关键词
- 支持按文档名称、标签搜索
- 实时显示搜索结果

**筛选文档**：
```
按文档类型筛选：
- 全部文档
- 方案文档
- 产品手册
- 竞品分析
- 其他

按上传时间筛选：
- 最近7天
- 最近30天
- 最近3个月
- 全部时间
```

**排序方式**：
- 按上传时间（默认，最新在前）
- 按文件大小
- 按文档名称（A-Z）

**列表信息**：
| 列名 | 说明 |
|------|------|
| 文档名称 | 点击可查看详情 |
| 文档类型 | 方案/手册/分析等 |
| 文件大小 | KB/MB |
| 上传时间 | YYYY-MM-DD HH:MM |
| 标签 | 文档标签（可多个） |
| 操作 | 查看/下载/删除 |

### 3.3 查看文档详情

**操作步骤**：
1. 在文档列表中点击文档名称
2. 进入文档详情页

**详情页内容**：

**基本信息**：
- 文档名称
- 文档类型
- 文件大小
- 上传时间
- 上传者
- 标签
- 备注说明

**内容预览**：
- 显示文档前1000字内容
- 支持关键词高亮

**关联方案**：
- 显示引用此文档的方案列表
- 点击可跳转到方案详情

**操作按钮**：
- 下载原文件
- 编辑文档信息
- 删除文档

### 3.4 删除文档

**操作步骤**：
1. 在文档列表中点击"删除"按钮
2. 弹出确认对话框："确定删除此文档吗？删除后无法恢复"
3. 点击"确定"
4. 系统删除文档文件和向量数据

**注意**：
- ⚠️ 删除操作不可恢复
- ⚠️ 删除文档不会影响已生成的方案
- ⚠️ 如文档被多个方案引用，建议谨慎删除

## 4. 方案生成

### 4.1 快速生成方案

**适用场景**：快速响应客户需求，自动生成方案初稿。

**操作步骤**：

#### 步骤1：创建方案

1. 点击左侧菜单"方案管理"
2. 点击"创建方案"按钮
3. 进入方案创建页面

#### 步骤2：填写基本信息

```
客户名称（必填）：例如"XX银行"、"XX证券"
项目名称（必填）：例如"智能营销平台建设"
预算（选填）：例如 1,000,000 元
时间周期（选填）：例如"6个月"
```

#### 步骤3：输入需求描述

在"需求描述"文本框中详细描述客户需求，**建议包含**：

```
1. 业务背景：客户行业、规模、现状
2. 核心需求：要解决的主要问题
3. 技术要求：性能、安全、兼容性等
4. 预期目标：量化的业务目标
```

**示例需求描述**：

```
【客户背景】
XX银行是一家全国性股份制商业银行，资产规模5000亿，员工1万人。
目前营销工作主要依靠人工，效率低、成本高、效果难以评估。

【核心需求】
希望构建一套基于AI的智能营销平台，实现：
1. 客户画像分析：基于大数据分析客户行为偏好
2. 精准营销推荐：AI推荐适合的金融产品
3. 营销效果评估：实时监控营销活动ROI

【技术要求】
- 支持100万+客户数据处理
- API响应时间<100ms
- 支持私有化部署
- 符合金融行业数据安全规范

【预期目标】
- 提升营销转化率30%以上
- 降低人工营销成本50%
- 6个月内上线投产
```

**需求描述技巧**：
- ✅ 越详细越好（建议200字以上）
- ✅ 包含量化指标
- ✅ 说明行业特点和约束条件
- ❌ 避免过于简短（< 50字）
- ❌ 避免模糊描述

#### 步骤4：生成方案

1. 检查信息填写完整
2. 点击"开始生成"按钮
3. 系统开始AI生成流程

**生成过程**（2-5分钟）：
```
[进度1/5] 分析需求描述...
[进度2/5] 搜索相似历史方案...
[进度3/5] 检索相关知识库...
[进度4/5] AI生成方案章节...
[进度5/5] 保存方案...
```

4. 生成完成后自动跳转到方案详情页

#### 步骤5：查看生成结果

方案包含以下章节：

| 章节 | 内容说明 | 字数 |
|------|---------|------|
| **执行摘要** | 方案核心亮点，高层汇报用 | 200-300字 |
| **需求分析** | 客户需求详细分析 | 500-800字 |
| **解决方案** | 技术方案、功能设计 | 1000-1500字 |
| **技术架构** | 系统架构、技术选型 | 800-1200字 |
| **实施计划** | 项目进度、里程碑 | 500-800字 |
| **风险评估** | 主要风险和应对措施 | 400-600字 |
| **报价说明** | 成本估算、报价明细 | 300-500字 |

### 4.2 编辑与完善方案

**重要提示**：AI生成的方案是初稿，必须人工审核和完善！

**操作步骤**：

1. 在方案详情页，点击"编辑方案"按钮
2. 进入编辑模式（富文本编辑器）
3. 编辑各个章节内容

**编辑器功能**：

**文本格式**：
- 加粗、斜体、下划线
- 标题（H1-H6）
- 有序列表、无序列表
- 引用、代码块

**插入元素**：
- 插入图片（支持上传或URL）
- 插入表格
- 插入链接

**快捷键**：
```
Ctrl + B: 加粗
Ctrl + I: 斜体
Ctrl + K: 插入链接
Ctrl + S: 保存
Ctrl + Z: 撤销
Ctrl + Y: 重做
```

**编辑技巧**：

**执行摘要优化**：
```
优化前：本项目旨在为XX银行构建智能营销平台...
优化后：通过AI大数据技术，帮助XX银行实现营销转化率提升30%，
       人工成本降低50%，预计6个月投产，投资回报期12个月。
       
技巧：突出核心价值、量化指标、时间节点
```

**需求分析优化**：
```
补充内容：
- 客户访谈记录
- 痛点详细描述
- 竞品对比分析
- 行业趋势分析
```

**解决方案优化**：
```
补充内容：
- 产品截图
- 功能演示视频链接
- 成功案例
- 技术优势说明
```

**技术架构优化**：
```
补充内容：
- 架构图（使用draw.io绘制后插入）
- 技术选型理由
- 性能指标说明
- 安全机制说明
```

**实施计划优化**：
```
细化内容：
- 详细的里程碑时间表
- 每个阶段的交付物
- 人员配置计划
- 风险缓解措施
```

**报价说明优化**：
```
补充内容：
- 详细的报价明细表
- 付款方式说明
- 售后服务说明
- 可选服务清单
```

4. 编辑完成后点击"保存"按钮

### 4.3 方案导出

**功能说明**：将方案导出为Word、Excel、PDF格式，用于提交客户或内部审批。

#### 导出Word文档

**操作步骤**：
1. 在方案详情页，点击"导出"按钮
2. 选择"导出为Word (.docx)"
3. 系统生成Word文档（3-5秒）
4. 浏览器自动下载文件

**文件名格式**：
```
方案_{客户名称}_{项目名称}_{日期}.docx

示例：
方案_XX银行_智能营销平台建设_20251108.docx
```

**Word文档特点**：
- 自动生成封面（含客户名称、项目名称、日期）
- 自动生成目录（可点击跳转）
- 专业的文档样式（标题、正文、表格）
- 页眉页脚（含页码）

**导出后处理建议**：
```
1. 打开Word文档检查格式
2. 添加公司Logo到封面
3. 调整页边距和行距
4. 插入架构图和截图
5. 检查错别字
6. 转为PDF（File → Save As → PDF）
```

#### 导出Excel报价单

**操作步骤**：
1. 点击"导出" → "导出报价单 (.xlsx)"
2. 系统生成Excel格式的报价明细表
3. 浏览器自动下载

**Excel内容**：
- 项目概述
- 明细报价表（序号、项目名称、单价、数量、金额）
- 总价汇总
- 付款条款

#### 导出PDF（需配置）

**操作步骤**：
1. 点击"导出" → "导出为PDF (.pdf)"
2. 系统先生成Word再转换为PDF
3. 下载PDF文件

**注意**：此功能需要后端配置LibreOffice，如未配置可先导出Word再手动转PDF。

## 5. 知识库管理

### 5.1 添加知识条目

**功能说明**：手动添加产品知识、技术要点、行业最佳实践等，增强AI生成效果。

**操作步骤**：

1. 点击左侧菜单"知识库"
2. 点击"添加知识"按钮

## 7. 常见操作FAQ

### Q1: 方案生成失败怎么办？

**可能原因**：
1. 智谱AI API密钥失效或额度用完
2. 网络连接问题
3. 需求描述过于简短（< 50字）
4. 服务器负载过高

**解决方法**：
```
步骤1：检查网络连接
- ping通服务器地址

步骤2：检查需求描述
- 确保需求描述至少200字
- 包含背景、需求、目标

步骤3：联系管理员
- 检查AI API配置
- 检查服务器日志

步骤4：重试
- 等待5分钟后重新生成
```

### Q2: 如何提升AI生成质量？

**最佳实践**：

**1. 上传高质量历史方案**
```
建议：
- 至少上传10份优秀方案
- 涵盖不同行业和场景
- 方案质量越高，AI学习效果越好
```

**2. 完善知识库**
```
建议添加：
- 所有产品的详细介绍
- 技术方案和架构文档
- 成功案例和实施经验
- 行业最佳实践
```

**3. 详细描述需求**
```
需求描述至少包含：
- 客户背景（行业、规模）
- 业务痛点（要解决的问题）
- 功能需求（具体要求）
- 技术要求（性能、安全）
- 预期目标（量化指标）
- 约束条件（预算、时间）

建议字数：200-500字
```

**4. 人工完善**
```
AI生成只是初稿，必须人工：
- 审核内容准确性
- 补充细节和数据
- 调整语气和风格
- 添加图表和截图
- 修正错误和不合理之处
```

### Q3: 方案内容如何保密？

**安全措施**：

**1. 本地存储**
- 所有方案存储在公司内部服务器
- 不上传到任何云服务
- 数据库加密存储

**2. AI调用脱敏**
- 调用智谱AI时自动脱敏客户信息
- 不发送敏感数据（如报价、合同）
- 仅发送需求描述用于生成

**3. 访问控制**
- JWT身份认证
- 用户只能查看自己的方案
- 管理员审计日志

**4. 私有化部署**
- 支持完全私有化部署
- 可配置本地AI模型（不调用外部API）
- 符合金融行业数据安全规范

### Q4: 可以修改AI生成的内容吗？

**答案**：可以，而且强烈建议修改！

**原因**：
- AI生成的是基于历史方案的初稿
- 可能包含通用性内容
- 需要根据客户实际情况调整

**必须修改的内容**：
1. **客户专属信息**
   - 客户名称、项目背景
   - 客户痛点和需求
   - 客户IT现状

2. **项目具体信息**
   - 具体的功能需求
   - 技术参数和指标
   - 实施计划和时间表
   - 报价明细

3. **添加素材**
   - 产品截图
   - 架构图
   - 流程图
   - 成功案例

### Q5: 如何批量导出方案？

**操作步骤**：

1. 进入方案列表页
2. 勾选需要导出的方案（可多选）
3. 点击"批量操作" → "批量导出"
4. 系统打包所有方案为ZIP文件
5. 浏览器自动下载ZIP文件

**ZIP文件结构**：
```
方案_批量导出_20251108.zip
├── 方案_XX银行_智能营销平台_20251108.docx
├── 方案_YY证券_交易系统_20251107.docx
├── 方案_ZZ保险_理赔系统_20251106.docx
└── 导出清单.txt
```

### Q6: 忘记保存了怎么办？

**自动保存功能**：
- 系统每5分钟自动保存草稿
- 编辑方案时自动保存

**恢复草稿**：
1. 进入方案编辑页
2. 如有未保存内容，系统提示"发现自动保存的草稿"
3. 点击"恢复草稿"
4. 继续编辑

### Q7: 可以复制已有方案吗？

**可以**，支持方案复制功能。

**操作步骤**：
1. 在方案详情页，点击"更多操作" → "复制方案"
2. 修改客户名称和项目名称
3. 点击"确定"
4. 系统创建新方案（内容与原方案相同）
5. 编辑新方案内容

**适用场景**：
- 同一客户的多个项目
- 相似需求的不同客户
- 方案版本管理
