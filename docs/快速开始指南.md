# 金融售前方案辅助编写系统 - 快速开始指南

## 🚀 5分钟快速上手

欢迎来到金融售前方案辅助编写系统！本指南将帮助您在最短时间内了解和使用系统的核心功能。

## 📋 前置准备

### ✅ 系统要求
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **网络**: 稳定的互联网连接
- **权限**: 有效的系统账户（联系管理员获取）

### 🔑 获取访问权限
1. 联系系统管理员创建账户
2. 获取登录凭据（用户名/密码）
3. 确保账户已激活并具有相应权限

---

## 🎯 第一步：登录系统

### 1. 访问系统
```
打开浏览器，输入系统地址：
开发环境: http://localhost:3000
生产环境: https://your-domain.com
```

### 2. 输入登录信息
```
用户名: [您的用户名]
密码: [您的密码]
验证码: [如有验证码，输入显示的数字]
```

### 3. 首次登录设置
- 修改初始密码（系统会提示）
- 设置个人信息（姓名、部门等）
- 配置偏好设置（语言、时区等）

---

## 📄 第二步：上传您的第一个文档

### 🚀 快速上传
1. **点击"文档管理"** - 在主菜单中选择文档管理
2. **选择"上传文档"** - 点击上传按钮
3. **拖拽文件** - 将文档拖拽到上传区域
4. **设置属性** - 填写文档标题、分类、标签
5. **确认上传** - 点击确认完成上传

### 📋 支持的文档类型
- **Word文档**: .doc, .docx
- **PDF文件**: .pdf
- **Excel表格**: .xls, .xlsx
- **PowerPoint**: .ppt, .pptx
- **文本文件**: .txt, .md

### 💡 上传技巧
- **批量上传**: 可同时选择多个文件
- **智能分类**: 系统会自动建议分类
- **标签管理**: 添加相关标签便于检索
- **预览确认**: 上传前可预览文档内容

---

## 🤖 第三步：生成您的第一个方案

### 🎯 方案生成流程

#### 1. 进入方案生成页面
- 点击主菜单中的"方案生成"
- 选择"新建方案"按钮

#### 2. 输入客户需求
```
客户行业: [选择行业，如：银行、保险、证券]
项目类型: [选择类型，如：风控系统、数据平台]
预算范围: [选择预算区间]
时间要求: [选择交付时间]
需求描述: [详细描述客户需求]
```

#### 3. 选择生成模式
- **🤖 AI智能生成**: 完全由AI自动生成
- **📋 模板生成**: 基于选定模板生成
- **🔍 相似方案**: 基于历史方案生成

#### 4. 配置生成参数
- **方案长度**: 简洁/标准/详细
- **技术深度**: 概述/标准/深入
- **商务比重**: 技术导向/平衡/商务导向
- **语言风格**: 正式/专业/创新

#### 5. 生成并查看结果
- 点击"生成方案"按钮
- 等待AI处理完成（通常30-60秒）
- 查看生成的方案内容
- 进行必要的编辑和调整

---

## 📊 第四步：使用模板功能

### 🎨 浏览模板库
1. **访问模板中心** - 点击"模板管理"
2. **选择模板类型** - 方案模板/报价模板/合同模板
3. **浏览模板库** - 查看可用模板
4. **预览模板** - 点击模板查看详细内容

### 🔧 使用模板生成方案
1. **选择合适模板** - 根据项目类型选择模板
2. **填写模板变量** - 输入客户名称、项目名称等变量
3. **生成方案** - 基于模板生成完整方案
4. **自定义编辑** - 根据需要调整内容

### 📋 创建自定义模板
1. **进入模板编辑器** - 选择"新建模板"
2. **设计模板结构** - 定义章节和段落
3. **设置模板变量** - 添加可替换的变量
4. **保存模板** - 命名并保存到模板库

## ⚠️ 常见问题和解决方案

### 🔑 登录问题
**问题**: 无法登录系统
**解决**:
- 检查用户名和密码是否正确
- 确认账户是否已激活
- 联系管理员重置密码
- 清除浏览器缓存后重试

### 📄 上传问题
**问题**: 文档上传失败
**解决**:
- 检查文件格式是否支持
- 确认文件大小未超过限制
- 检查网络连接是否稳定
- 尝试重新登录后上传

### 🤖 生成问题
**问题**: AI生成内容不符合预期
**解决**:
- 优化输入的需求描述
- 调整生成参数设置
- 选择合适的生成模式
- 使用更具体的模板

### 📤 导出问题
**问题**: 导出格式不正确
**解决**:
- 检查导出设置是否正确
- 确认浏览器版本兼容性
- 尝试使用不同格式导出
- 检查系统权限设置
